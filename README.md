# SmartPolls UI (React + Vite + Tailwind)

A lightweight poll platform UI:

- Create polls (validated form)
- Vote with basic duplicate prevention (per browser)
- View results with charts (Recharts)
- Dashboard to manage polls

## Tech Stack

- React + TypeScript
- Vite
- Tailwind CSS
- React Router
- react-hook-form + zod
- Recharts
- Radix Dialog
- sonner (toasts)

## Run locally

```bash
npm install
npm run dev
npm run lint
npm run format
```
